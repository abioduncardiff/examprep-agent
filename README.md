# ExamPrep Agent

ExamPrep Agent is an AI agent API that helps students prepare for exams.

## Project idea

Many students struggle to organise revision before exams. They may not know which topics to prioritise, how to practise, or how to identify weak areas.

ExamPrep Agent helps students by creating a personalised study plan, generating practice quizzes, marking answers, and recommending what to study next.

## Agent goal

The goal of this agent is to help a student prepare for an exam by:

- collecting exam information
- identifying weak topics
- creating a study plan
- generating quiz questions
- marking quiz answers
- giving feedback and next study recommendations

```text
